## Getting Started
Install dependencies.

```bash
$ npm install
```

Build

```bash
$ npm run build
$ npm run build:dll
```


Start srr server.

```bash
$ node server/index.js
```

Start dev

```bash
$ npm run build:dll
$ npm start
```
